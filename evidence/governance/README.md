# Historical delivery provenance

These records preserve earlier ownership decisions, assignments, authorization,
and SpecKit lifecycle attempts. They are immutable project history.

They no longer authorize implementation, assign current work, define current
status, or establish completion. Current ownership and delivery order are in
`docs/v2-delivery.md`; the product outcome is in
`docs/v2-completion-goal.md`. Ordinary code and reproducible verification
determine what exists.
