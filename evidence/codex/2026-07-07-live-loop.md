## Known caveats (honest)

- Headless Codex MCP tool calls currently require
  `--dangerously-bypass-approvals-and-sandbox`; `approval_policy=never`
  alone still auto-declines them. Broad flag — documented, needs a narrower
  codex-side knob when one exists.
- Rolling history is per-process; a runner restart starts blind (visible in
  this very run). Durable history is a possible follow-up.
- The interactive-session hook (`nunchi_prompt_gate_codex.py`) is
  offline-tested only; this run exercised the runner path.
